# ESGen
Repositório oficial do ESGen: Excel Sheet Generator, o gerador de planilhas de preços p/ o Excel.

## Descrição
Gerador de planilhas de compras automatizado via scraping.

## Versões
### [v1.0.0]!(builds/esgen-v1.0.0.exe)
- Criação da versão inicial (alfa) do Excel Sheet Generator (ESGen) com interface feita com Tkinter.
- **Features:**
  - Coleta automática de preços com suporte aos seguintes sites:
    - KaBuM;
    - Mercado Livre;
    - DetonaShop;
  - Entrada com quantidade personalizada (número após o link);
  - Geração de planilha Excel formatada, com as seguintes colunas:
    - Item;
    - Valor unitário;
    - Qtde;
    - Valor total;
    - Fornecedor;
    - Links;
  - Inclui total da compra;
  - Barra de Ajuda (com opção Sobre);
  - Barra de progresso;
  - Opção de abrir planilha assim que estiver pronta.
