# ESGen
Repositório oficial do ESGen: Excel Sheet Generator, o gerador de planilhas de preços p/ o Excel.

## Descrição
Gerador de planilhas de compras automatizado via scraping.

## Versões
### esgen-v1.0.0
- Criação da versão inicial (alfa) do Excel Sheet Generator (ESGen).
- **Features:**
  - Coleta automática de preços
  - Suporte aos seguintes sites:
    - KaBuM
    - Mercado Livre
    - DetonaShop
  - Entrada com quantidade personalizada
  - Geração de planilha Excel formatada, com as colunas
    - Item
    - Valor unitário
    - Qtde
    - Valor total
    - Fornecedor
    - Links
  - Total automático da compra
  - Interface gráfica (Tkinter)
  - Barra de Ajuda e Sobre
  - Barra de progresso
  - Opção de abrir planilha assim que estiver pronta
