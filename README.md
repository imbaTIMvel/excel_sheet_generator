# ESGen
Repositório oficial do ESGen: Excel Sheet Generator, o gerador de planilhas de preços p/ o Excel.

## Descrição
Gerador de planilhas de compras automatizado via scraping.

## Versões
### 1.0.0
- Coleta automática de preços
- Suporte aos seguintes sites:
  - KaBuM
  - Mercado Livre
  - DetonaShop
- Entrada com quantidade personalizada
- Geração de planilha Excel formatada
- Total automático da compra
- Interface gráfica (Tkinter)
